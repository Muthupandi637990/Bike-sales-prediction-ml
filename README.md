# Bike-sales-prediction-ml
Machine learning project to predict bike sales using Linear Regression, EDA and feature engineering.
# 🚲 Bike Sales Prediction using Machine Learning

## 📌 Project Overview
This project predicts bike sales using Machine Learning techniques. The goal is to identify key factors affecting sales and build a regression model for forecasting.

## 📊 Dataset
The dataset contains features such as:
- Price
- Seasonality
- Product features
- Sales data

## 🧠 Techniques Used
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Linear Regression Model

## 📈 Model Evaluation
- R² Score
- MAE
- RMSE

Achieved R² Score: **0.82**

## 🛠 Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## ▶️ How to Run
1. Install dependencies
   pip install -r requirements.txt

2. Run Jupyter Notebook
   jupyter notebook
